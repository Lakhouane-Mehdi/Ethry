# Ethry

A 2D top-down adventure game built with Godot 4.6 and C#.

## Features

- Player movement and animation
- Attack system with hitbox detection
- Pixel art style

## Getting Started

1. Open the project in Godot 4.6 (.NET version)
2. Build the C# solution
3. Run the project
